# SQL Essentials con Snowflake

Este curso introduce al participante en el mundo del análisis de datos mediante SQL aplicado directamente en Snowflake, una de las plataformas de datos en la nube más utilizadas en la industria. A través de ejercicios prácticos y casos reales, los estudiantes aprenderán a construir consultas desde cero, interpretar la estructura de las bases de datos, transformar información y extraer respuestas de negocio de manera eficiente. Al finalizar, el participante podrá realizar análisis exploratorios de datos de forma independiente dentro de un entorno moderno de data warehouse.
